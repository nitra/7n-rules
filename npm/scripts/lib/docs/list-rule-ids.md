## Огляд

Модуль перебирає директорії у `rules/<id>/`. Він виявляє каталоги, які містять файл `check.mjs`, оскільки відсутність цього файлу означає, що каталог є заглушкою або не є правилом. Модуль повертає відсортований список ідентифікаторів цих правил.

## Поведінка

1. Отримує список усіх підкаталогів у вказаній директорії правил.
2. Фільтрує цей список, залишаючи лише ті каталоги, які не є прихованими.
3. Для кожного каталогу перевіряє, чи існує у ньому файл `check.mjs`. Каталоги без цього файлу ігноруються.
4. Застосовує фільтр, якщо він був наданий, щоб залишити лише відповідний ID правила.
5. Сортує отримані ID правил в алфавітному порядку.
6. Повертає відсортований список ID правил.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
