## Огляд

Публічна функція `lint` перевіряє код на відповідність правилам, визначеним у конфігураційному файлі `.jscpd.json`. Відсутність цього файлу призводить до виведення користувачу рекомендації створити його за зразком, описаним у `js.mdc`.

## Поведінка

1. Викликає функцію `lint` для перевірки конфігурацій з використанням політик, визначених у файлі .jscpd.json.
2. Натомість повідомляє про відсутність файлу .jscpd.json, якщо він не знайдений, з рекомендацією створити його згідно з інструкціями в js.mdc.

## Публічний API

lint — Визначає політику виявлення проблем, пов'язаних із певними галузями знань (згенеровано обгорткою від codegen).
