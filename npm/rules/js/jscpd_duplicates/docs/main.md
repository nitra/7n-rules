## Огляд

Публічна функція `lint` аналізує вихідний код у вказаному кореневому каталозі, використовуючи `jscpd`, щоб виявляти дублікати та інші порушення стилю і логіки коду. Функція повертає код виходу, який інформує про успішне завершення аналізу чи виявлення проблем.

## Поведінка

1. Викликається функція `lint`, яка ініціює перевірку коду у вказаному кореневому каталозі з метою виявлення дублікатів.
2. Процес аналізу виконується за допомогою `jscpd` у середовищі `bunx`.
3. Функція повертає код завершення процесу: 0 у разі успішного виконання (без знайдених проблем) або інше число, що вказує на помилки.

## Публічний API

lint — виявляє дублікати коду за допомогою `bunx jscpd` у файлах JavaScript/JSCPD.
