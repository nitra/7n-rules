## Огляд

Ця функція здійснює аналіз коду відповідно до політики, визначеної у `lint-js.yml`. Вона сканує всі файли, окрім тих, що знаходяться в каталогах `.github` або `.git`, і виявляє стилістичні, логічні та синтаксичні порушення згідно з цією політикою.

## Поведінка

1. Викликається функція `lint`.
2. Система сканує файли, які не належать до каталогів .github або .git.
3. Система виконує аналіз, використовуючи політику, визначену у файлі .github/workflows/lint-js.yml, для визначення порушень.

## Публічний API

lint — виявляє ознаки, що можуть свідчити про політичні питання.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Свідомо пропускає шляхи: `.github`, `.git`.
