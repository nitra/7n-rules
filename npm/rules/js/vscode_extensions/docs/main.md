## Огляд

Перевіряє відповідність конфігурації розширення, спираючись на налаштування, визначені у файлі extensions.json. Якщо цей файл відсутній, інформує про необхідність додати рекомендації відповідно до конвенцій, описаних у js.mdc. При успішній валідації надає звіт, деталізуючи виявлені відхилення від визначеної конфігурації.

## Поведінка

1. Викликається функція `lint` для перевірки відповідності конфігурації розширення.
2. Перевірка базується на файлі `.vscode/extensions.json`.
3. Якщо файл `.vscode/extensions.json` відсутній, генерується повідомлення про те, що необхідно додати рекомендації з js.mdc.
4. При успішній перевірці, функція повертає уніфікований результат лінту зі списком виявлених порушень.

## Публічний API

lint — шукає порушення політики зони відповідальності (policy-concern).
