## Огляд

Аналізує проєкт на предмет невикористаних залежностей та експортів. Виконує операцію лише для читання, не змінюючи файлову систему чи базу даних. Публічна функція `lint` надає звіт про виявлені проблеми, який визначається за кодом виходу.

## Поведінка

1. Викликається функція `lint`.
2. Система аналізує проект на предмет невикористаних залежностей та експортів.
3. На основі збору виявлених проблем, система генерує звіт.
4. Якщо знайдено хоча б одну проблему, функція повертає значення 1; інакше — 0.
