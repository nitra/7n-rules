## Огляд

Цей файл перевіряє відповідність контенту вимогам MDC, включаючи політику та посилання. Він виконує повну перевірку, коли викликається як окрема утиліта командного рядка.

## Поведінка

1. Викликається функція `run` для виконання перевірки.
2. Виконання перевірки застосовує логіку, пов'язану з JS-занепокоєннями, політикою та посиланнями MDC.
3. Якщо скрипт виконується як окрема утиліта командного рядка, виконується повна логіка правила.

## Публічний API

run — виконує послідовність перевірок: застосовує вимоги, перевіряє аспекти JavaScript, застосовує політику та посилання на MDC.
lint — виконує легку перевірку коду, важкі операції перенесені в окремий модуль.
