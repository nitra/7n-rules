## Огляд

Ініціалізує процес через функцію `main`, а потім сканує робочу директорію для виявлення всіх каталогів з конфігурацією Tauri (`src-tauri/`). Для кожного знайденого каталогу перевіряє наявність файлу `.cargo/mutants.toml`, забезпечуючи його створення з канонічними налаштуваннями Tauri у разі відсутності. Якщо файл існує, він реєструє та додає відсутні канонічні ключі у конфігураційний файл, зберігаючи при цьому оригінальний вміст, і повідомляє про це через (tauri.mdc).

## Поведінка

Поведінка:

1. Викликає `main` для ініціалізації процесу.
2. Визначає всі каталоги, що містять конфігурацію Tauri (`src-tauri/`) у робочому просторі.
3. Для кожного знайденого каталогу перевіряє наявність файлу `.cargo/mutants.toml`.
4. Якщо файл відсутній, створює його, заповнюючи канонічними налаштуваннями Tauri. Повідомляє про створення файлу з маркером (tauri.mdc).
5. Якщо файл присутній, виявляє та реєструє відсутні канонічні ключі в конфігурації.
6. Якщо знайдені ключі відсутні, додає їх до існуючого файлу, зберігаючи原有 вміст. Повідомляє про додавання ключів з маркером (tauri.mdc).
7. Якщо ключі присутні, зберігає існуючу конфігурацію без змін.
