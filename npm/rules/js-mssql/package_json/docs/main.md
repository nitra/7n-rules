## Огляд

Функція `lint` аналізує конфігураційні файли `package.json` у поточній директорії та її підкаталогах. Вона перевіряє код на відповідність правилам, визначеним у `package.json`. Повертає структурований перелік виявлених порушень відповідності конфігурації.

## Поведінка

1. Визначає політичні занепокоєння, перевіряючи конфігураційні файли `package.json` у поточній робочій директорії та її підкаталогах.
