## Огляд

Модуль перевіряє відповідність даних логіці, пов'язаній із JS-зацікавленостями, політикою та посиланнями MDC. Він виконує перевірку, яка є еквівалентом команди `npx @nitra/cursor check <id>` при запуску як окрема утиліта командного рядка.

## Поведінка

1. Викликається функція `run` для виконання перевірки.
2. Виконання перевірки застосовує логіку, пов'язану з JS-зацікавленостями, політикою та посиланнями MDC.
3. Якщо скрипт запускається як окрема утиліта командного рядка, виконується повний еквівалент команди `npx @nitra/cursor check <id>`.

## Публічний API

run — виконує послідовність перевірок: застосовує вимоги, перевіряє логіку JavaScript, застосовує політику та посилання на MDC.
lint — виконує легку перевірку коду, складна логіка перенесена до окремого модуля.
