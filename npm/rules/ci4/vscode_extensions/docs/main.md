## Огляд

Перевіряє конфігурацію розширення VS Code, спираючись на налаштування, визначені у `extensions.json`. Використовуючи логіку функції `lint`, виявляє порушення конфігурації розширення, необхідні для коректної роботи системи. Повертає список знайдених порушень.

## Поведінка

1. Виконує перевірку конфігурацій розширення VS Code, спираючись на файл extensions.json.
2. Повертає результат лінту, включаючи виявлені порушення.
