Модуль виконує перевірку зображень. Він може стискати зображення, якщо це дозволено режимом, або лише перевіряти, які зображення потребують стиснення, якщо режим лише для читання. Результатом є код завершення інструменту стиснення або код помилки, якщо інструмент не запустився. Модуль перехоплює помилки, не кидаючи винятків назовні.

## Поведінка

1. Викликається `lint` для перевірки зображень.
2. Якщо вказано режим лише для читання, виконується перевірка наявності зображень, що потребують стиснення, без внесення змін.
3. Якщо режим не лише для читання, виконується стиснення зображень.
4. У разі помилки запуску інструменту стиснення, повертається код помилки.
5. У разі успішного виконання, повертається код завершення інструменту стиснення.

## Публічний API

lint — керує процесом стилізації зображень за допомогою `n-cursor lint image-compress`.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Перехоплює помилки і не пропускає винятків назовні (fail-safe).
- Кешує результати в межах одного прогону.
