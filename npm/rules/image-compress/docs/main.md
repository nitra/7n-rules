## Огляд

Модуль перевіряє відповідність скрипта логіці, пов'язаній з JS-зацікавленостями, політикою та посиланнями MDC. Перевірка запускається, коли скрипт працює як автономний клієнт.

## Поведінка

1. Викликається `run` для виконання перевірки.
2. Виконання перевірки застосовує логіку, пов'язану з JS-зацікавленостями, політикою та посиланнями MDC.
3. Якщо скрипт виконується як автономний клієнт, відбувається запуск правила.

## Публічний API

run — виконує повний цикл застосування правил: перевіряє відповідність вимогам, застосовує логіку, перевіряє політику та посилання на метадані.
lint — виконує легку перевірку коду, складні операції перенесено в окремий модуль.
