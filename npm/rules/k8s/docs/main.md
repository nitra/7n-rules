## Огляд

Цей файл реалізує логіку перевірки, що застосовує визначені політики MDC. Він виконує перевірку, коли запускається як незалежний клієнтський інструмент.

## Поведінка

1. Викликається функція `run` для виконання перевірки.
2. Виконання перевірки застосовує політики та посилання MDC.
3. Якщо скрипт виконується як окремий клієнтський інструмент, запускається повна реалізація правила.

## Публічний API

run — виконує послідовність дій: застосовує логіку, обробляє аспекти JavaScript, перевіряє політику та посилання MDC.
lint — виконує перевірку синтаксису та стилю коду.
