Модуль виконує перевірку правил, пов'язаних із JS-занепокоєннями, політикою та посиланнями MDC. При виклику з іншого коду він запускає перевірку у режимі бібліотеки. При запуску як окрема програма він виконує повний цикл валідації, включаючи завантаження конфігурації та збір підсумків.

## Поведінка

1. Викликає функцію `run` для запуску перевірки в режимі бібліотеки. Це виконує застосування правил, пов'язаних з JS-занепокоєннями, політикою та посиланнями MDC.
2. Якщо скрипт запускається як окрема програма, виконується оркестрація командного рядка. Це забезпечує повний запуск правила, включаючи завантаження конфігурації та підсумок.

## Публічний API

run — виконує послідовність дій: застосовує правило, обробляє JS-занепокоєння, перевіряє політику та посилання MDC.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Кешує результати в межах одного прогону.
