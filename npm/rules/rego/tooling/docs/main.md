## Огляд

Викликаючи `main`, система перевіряє наявність файлу `.regal/config.yaml` у корені проєкту. Якщо файл існує, це підтверджує відповідність конфігурації вимогам `rego.mdc`, включаючи маркери повідомлень (згадай у Поведінці).
