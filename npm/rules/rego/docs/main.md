## Огляд

Цей файл ініціює перевірку відповідності логіки, пов'язаної з JS-занепокоєннями, політиками та посиланнями MDC. Перевірка здійснюється через виклик функції `run`.

## Поведінка

1. Викликати `run` для виконання перевірки. Це застосовує логіку, пов'язану з JS-занепокоєннями, політиками та посиланнями MDC.
2. Якщо скрипт виконується як окрема командна лінія, виконати повний еквівалент команди `npx @nitra/cursor check <id>` для даного правила.

## Публічний API

run — виконує послідовність перевірок: застосовує логіку, перевіряє аспекти JavaScript, застосовує політику та посилання на метрики.
lint — виконує перевірку стилю коду.
