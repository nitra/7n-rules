## Огляд

Модуль перевіряє відповідність конфігурації та білих списків. При запуску як окрема утиліта командного рядка, він завантажує конфігурацію для формування підсумків після повного процесу перевірки.

## Поведінка

1. Викликати функцію `run` для запуску правила. Це виконує перевірку, яка включає застосування логіки, пов'язаної з JS-занепокоєннями, політики та посиланнями MDC.
2. Якщо скрипт виконується як окрема утиліта командного рядка, він запускає повний процес перевірки правил. Це включає завантаження конфігурації, застосування білих списків та формування підсумків.

## Публічний API

run — виконує послідовність перевірок: застосовує правила, аналізує JS-занепокоєння, перевіряє політику та посилання MDC.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Кешує результати в межах одного прогону.
