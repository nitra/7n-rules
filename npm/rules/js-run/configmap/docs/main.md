## Огляд

Валідує конфігураційні файли Kubernetes (`configmap.yaml`), застосовуючи набір політик безпеки, визначених рушієм `rego`. Виявляє потенційні порушення конфігурації та надає уніфікований звіт про всі знайдені недоліки.

## Поведінка

1. Викликає `lint` для перевірки конфігураційних файлів у форматі `configmap.yaml`, розташованих у директоріях, що відповідають структурі Kubernetes (`k8s/*/configmap.yaml`).
2. Виконує оцінку політики, використовуючи рушій `rego`, для виявлення можливих порушень у перелічених файлах.
3. Повертає уніфікований результат перевірки, який містить список виявлених порушень.

## Публічний API

lint — перевіряє, чи код відповідає політиці, визначеній у `codegen-обгортці`.
