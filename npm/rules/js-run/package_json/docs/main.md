## Огляд

Файл виконує дельта-лінтування конфігураційного файлу `package.json` у поточній директорії та піддиректоріях. Він порівнює вміст `package.json` з політичними вимогами, визначеними двигуном `rego`. Результатом є уніфікований список виявлених невідповідностей.

## Поведінка

1. Викликається функція `lint`.
2. Система аналізує конфігураційний файл `package.json` у поточній директорії та всіх піддиректоріях.
3. Система перевіряє відповідність конфігурації встановленим політичним вимогам (на основі двигуна `rego`).
4. Повертається уніфікований результат, що містить список виявлених порушень.

## Публічний API

lint — виявляє порушення, пов'язані з політикою `concern` (генерується за допомогою `codegen-обгортки`).
