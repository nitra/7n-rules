## Огляд

Перевіряє відповідність JS-зацікавленостей, політиці та посиланням MDC. Виконує повне правило при запуску як окрема утиліта командного рядка.

## Поведінка

1. Викликається функція `run` для виконання перевірки.
2. Виконання перевірки застосовує логіку, пов'язану з JS-зацікавленостями, політикою та посиланнями MDC.
3. Якщо скрипт виконується як окрема утиліта командного рядка, виконується повна логіка правила.

## Публічний API

run — виконує послідовність дій: застосовує логіку, обробляє JS-залежності, перевіряє політику та посилання MDC.
lint — виконує перевірку коду, делегуючи складну реалізацію зовнішньому модулю.
