## Огляд

Конфігурація системи перевіряється за допомогою функції `lint`. Ця функція забезпечує валідацію конфігурацій, необхідних для роботи системи, порівнюючи їх із політиками, визначеними в рего-двигуні. Перевірка спирається на конфігураційні дані з файлу extensions.json та враховує вимоги, зазначені в graphql.mdc.

## Поведінка

1. Викликається функція `lint`.
2. Програма перевіряє наявність конфігурацій у файлі `.vscode/extensions.json`.
3. Якщо файл `.vscode/extensions.json` відсутній, генерується повідомлення, що рекомендує додати розширення згідно з правилом `graphql.mdc`.
4. Якщо файл знайдено, виконується перевірка конфігурацій на відповідність політиці, визначеній у рего-двигуні.

## Публічний API

lint — виявляє проблеми, пов'язані з політикою конфігурації.
