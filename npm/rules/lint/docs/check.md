Модуль виконує правило прогону. Публічна функція `run` повертає результат виконання цього правила. При запуску з командного рядка, процес завершується кодом, отриманим від виконання правила.

## Поведінка

1. Викликати стандартне правило для прогону.
2. Повертати результат цього прогону.
3. Якщо код виконується як CLI, завершити процес з кодом, отриманим від виконання правила.

## Публічний API

run — виконує правило `lint` в оркестраторі, повертаючи успіх, якщо немає конфігурацій для перевірки.
