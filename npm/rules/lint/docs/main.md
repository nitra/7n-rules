## Огляд

Модуль ініціює виконання правил, визначених у конфігурації, передаючи їх до механізму стандартного правила. У режимі командного рядка процес завершується після виконання правила через механізм CLI.

## Поведінка

1. Викликається функція `run` для виконання правила.
2. Виконання правила відбувається шляхом передачі директорії правила до механізму виконання стандартного правила.
3. Якщо код виконується як CLI, викликається механізм виконання правила CLI, який завершує процес.

## Публічний API

run — виконує правило `lint` в lint-оркестраторі, повертаючи успіх, якщо немає конфігурацій для перевірки.
