## Огляд

Файл запускає перевірку відповідності коду встановленим політикам, використовуючи конфігурацію з package.json. Він аналізує зміст коду на підставі правил Rego, визначаючи його відповідність політичним вимогам. Результатом цього аналізу є єдиний звіт про знайдені порушення.

## Поведінка

1. Ініціює перевірку політики, використовуючи `package.json` як конфігураційний файл.
2. Визначає політичну зацікавленість (`policy-concern`) з використанням двигуна `rego`.
3. Повертає уніфікований результат перевірки з переліком знайдених порушень.
