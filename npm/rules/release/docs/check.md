Модуль ініціює виконання перевірок правил у системі. Під час виконання він використовує MDC-референси для валідації. При запуску через командний інтерфейс, він запускає оркестрацію CLI для виконання цих правил.

## Поведінка

1. Викликається функція `run` для запуску перевірки правил.
2. Перевірка правил виконується у режимі, що включає застосування політики та посилання на MDC-референси.
3. Якщо код виконується як командний інтерфейс, запускається оркестрація CLI для виконання правил.

## Публічний API

run — виконує послідовність дій: застосовує правило, обробляє JS-залежності, перевіряє політику та посилання MDC.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Кешує результати в межах одного прогону.
