## Огляд

Цей файл перевіряє відповідність JS-зацікавленості вимогам політики та посилань MDC. При запуску як окрема утиліта командного рядка ініціюється повне виконання цієї перевірки.

## Поведінка

1. Викликається функція `run` для виконання перевірки правила.
2. Виконання перевірки включає застосування логіки, пов'язаної з JS-зацікавленістю, політикою та посиланнями MDC.
3. Якщо скрипт виконується як окрема утиліта командного рядка, запускається повна логіка правила.

## Публічний API

run — виконує послідовність перевірок: застосовує логіку, перевіряє аспекти JavaScript, застосовує політику та посилання MDC.
lint — виконує легку перевірку коду, важкі операції перенесені в окремий модуль.
