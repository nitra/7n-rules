## Огляд

Огляд
Код перевіряє відповідність структури встановленим політикам, використовуючи публічну функцію `lint` для оцінки. Також здійснюється перевірка налаштувань конфігурації, які беруться з `settings.json` середовища розробки.

Поведінка
1. Викликає функцію `lint` для оцінки відповідності політики.
2. Перевіряє наявність файлу `.vscode/settings.json`, який є джерелом конфігурації, на яку спирається код.
3. У випадку відсутності конфігурації, яка відповідає за автоматичне форматування у робочих процесах, генерується повідомлення, що посилається на `ga.mdc`.

## Поведінка
