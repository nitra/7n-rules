## Огляд

Цей модуль виконує детерміновану генерацію конфігураційного файлу `.licensee.json` у корені проекту. Генерація відбувається через виконання канонічної команди `licensee --init`, що ініціалізує файл `.licensee.json` для забезпечення метаданих ліцензування проекту. Це механізм для встановлення початкової конфігурації.

## Поведінка

Поведінка:
1. Перевіряється, чи існують порушення, пов'язані з відсутністю конфігурації `licensee` у контексті.
2. Якщо порушення виявлено, викликається команда `bun x licensee --init --production --quiet` у поточному робочому каталозі для створення або ініціалізації файлу `.licensee.json`.
3. Результат виконання команди фіксується, а файл `.licensee.json` позначається як змінений.
4. Якщо порушень немає, жодних дій не виконується.

## Гарантії поведінки

- Пише лише `.licensee.json`; за відсутності `bun` у PATH — no-op.
