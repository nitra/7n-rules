Цей модуль виконує статичний аналіз коду. Він запускає лінтер для виявлення стилістичних помилок, проводить перевірку структури залежностей через Composer, виконує сухий прогін `php-cs-fixer` та аналіз типу за допомогою `phpstan`/`psalm`.

## Поведінка

1. Викликає стандартний лінтер для перевірки коду.
2. Виконує аудит Composer.
3. Запускає перевірку коду за допомогою php-cs-fixer у режимі сухого прогону.
4. Запускає перевірку коду за допомогою phpstan/psalm.

## Публічний API

lint — Запускає перевірки коду: аналіз залежностей (composer audit), форматування коду (php-cs-fixer) та статичний аналіз (phpstan/psalm).
Структурні php.mdc-перевірки — Перевіряє відповідність структури файлів конформності.
