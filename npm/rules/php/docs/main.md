## Огляд

Модуль перевіряє відповідність конфігурації логіці JS-зацікавленостей, політиці та посиланням MDC. Він забезпечує повну валідацію, що відповідає функціоналу, який надає команда `npx @nitra/cursor check <id>` при використанні як окрема утиліта командного рядка.

## Поведінка

1. Викликається функція `run` для виконання перевірки.
2. Виконання перевірки застосовує логіку, пов'язану з JS-зацікавленостями, політикою та посиланнями MDC.
3. Якщо скрипт виконується як окрема утиліта командного рядка, виконується повний еквівалент команди `npx @nitra/cursor check <id>`.

## Публічний API

run — виконує послідовність перевірок: застосовує логіку, аналізує JS-залежності, перевіряє політику та посилання на MDC.
lint — виконує легкий аналіз коду, складні перевірки делегує зовнішньому модулю.
