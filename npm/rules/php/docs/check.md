Модуль виконує перевірку файлів PHP відповідно до визначених політик. Він застосовує ці політики для валідації відповідності посилань MDC у файлах. Публічна функція `run` ініціює цей процес. Результат виконання правил визначає код виходу системи.

## Поведінка

1. Викликається функція `run`.
2. Виконується логіка, що застосовує правило до файлів PHP, перевіряючи відповідність політикам та посиланням MDC.
3. При запуску як окремий інструмент (CLI), виконується оркестрація правил.
4. При завершенні роботи, код виходу встановлюється на основі результату виконання правил.

## Публічний API

run — Виконує послідовність перевірок: застосування правил, перевірку JS-зацікавлень, відповідність політиці та посилання на MDC.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Кешує результати в межах одного прогону.
