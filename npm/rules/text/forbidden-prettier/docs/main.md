## Огляд

Огляд: Перевіряє, чи відповідає конфігураційна структура проєкту встановленим стандартам форматування, скануючи кореневу директорію на наявність файлів конфігурації Prettier, зокрема `.prettierrc.json`.

Поведінка:
Викликається `main` для запуску перевірки.
Система сканує кореневу директорію проєкту на наявність конфігураційних файлів, що стосуються Prettier.
Якщо виявлено заборонений конфігураційний файл Prettier у корені, система видає повідомлення з маркуванням (text.mdc) про необхідність корекції, оскільки необхідно використовувати oxfmt.
Якщо жодних заборонених файлів Prettier не знайдено, перевірка вважається успішною.

## Поведінка

Поведінка:

1. Викликається `main` для запуску перевірки.
2. Система сканує кореневу директорію проєкту на наявність конфігураційних файлів, що стосуються Prettier (зокрема, `.prettierrc.json`).
3. Якщо виявлено будь-який заборонений файл Prettier у корені, відбувається відхилення з повідомленням, що необхідно використовувати oxfmt, з посиланням на `text.mdc`.
4. Якщо жодних заборонених файлів Prettier не знайдено, перевірка вважається успішною.
