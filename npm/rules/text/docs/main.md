## Огляд

Модуль порівнює JavaScript-зацікавленості, політику та посилання MDC. Він гарантує повне виконання відповідного правила при запуску як окрема утиліта командного рядка.

## Поведінка

1. Викликається функція `run` для виконання перевірки.
2. Виконання перевірки застосовує логіку, пов'язану з JS-зацікавленостями, політикою та посиланнями MDC.
3. Якщо скрипт виконується як окрема утиліта командного рядка, виконується повний запуск правила.

## Публічний API

run — виконує послідовність перевірок: застосовує логіку, перевіряє відповідність JS-зацікавленостям, застосовує політику та посилання MDC.
lint — виконує легку перевірку коду, складна логіка перенесена в окремий модуль.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
