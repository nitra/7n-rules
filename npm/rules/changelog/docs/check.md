Модуль виконує оркестрацію перевірки при запуску як утиліта командного рядка. Він завантажує конфігурацію, перевіряє білий список та підсумовує результати. Перевірка виконується з використанням контексту прогону, що включає кешування у межах цього прогону.

## Поведінка

1. Викликається функція `run` для запуску перевірки.
2. Перевірка виконується з використанням контексту прогону, що може включати кеш.
3. Якщо скрипт виконується як окрема утиліта командного рядка, запускається повна оркестрація правила.
4. Оркестрація включає завантаження конфігурації, перевірку білого списку та підсумок.
5. Результат виконання повертає код виходу процесу.

## Публічний API

run — Запускає послідовність перевірок: застосування правил, аналіз JS-зацікавленостей, перевірку політик та посилання на MDC.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Кешує результати в межах одного прогону.
