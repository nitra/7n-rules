## Огляд

`patterns` описує узгоджені шаблони для `settings.json` і визначає, які значення в ньому мають відповідати очікуваному формату. Це потрібно, щоб конфіг лишався передбачуваним і однаково трактувався в різних місцях, де його використовують.

## Поведінка

1. `patterns` описує набір правил для приведення `.vscode/settings.json` до узгодженого стану з шаблонним стилем проєкту.
2. На основі цього набору система знаходить конфігурацію `settings.json` і підтягує її в механізм виправлення, щоб зберігати єдиний формат налаштувань у робочому середовищі.
3. Результат роботи `patterns` використовується як джерело стандарту для перевірки та вирівнювання вмісту `settings.json` без ручного втручання.
4. Модуль не змінює файлову систему чи бази даних; він лише надає опис поведінки для подальшого застосування.
