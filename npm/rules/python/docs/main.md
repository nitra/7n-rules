## Огляд

Цей файл перевіряє відповідність контенту вимогам JS-зацікавленостей, політики та посилань MDC. Він реалізує повний набір правил, що виконується як окрема утиліта командного рядка.

## Поведінка

1. Викликається функція `run` для виконання перевірки.
2. Виконання перевірки застосовує логіку, пов'язану з JS-зацікавленостями, політикою та посиланнями MDC.
3. Якщо скрипт виконується як окрема утиліта командного рядка, виконується повна логіка правила.

## Публічний API

run — виконує послідовність перевірок: застосовує логіку, перевіряє відповідність JS-зацікавленостям, застосовує політику та посилання MDC.
lint — виконує перевірку синтаксису та стилю, делегуючи складну реалізацію зовнішньому модулю.
