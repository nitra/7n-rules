## Огляд

Валідує конфігураційні файли Kubernetes у шаблоні `**/k8s/**/hc.yaml` на відповідність політиці, визначеній у наданих конфігураційних файлах. Генерує єдиний звіт про виявлені порушення.

## Поведінка

1. Викликає перевірку політики для файлів, що відповідають шаблону `**/k8s/**/hc.yaml`.
2. Визначає, що логіку перевірки здійснює двигун Rego.
3. Повертає уніфікований результат перевірки, який містить список знайдених порушень.

## Публічний API

lint — Виявляє політики, пов'язані з проблемами, визначені у коді.
