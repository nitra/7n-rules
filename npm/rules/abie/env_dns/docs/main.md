## Поведінка

Поведінка:

1. Визначає корінь репозиторію.
2. Зчитує конфігурацію ігнорування шляхів.
3. Збирає файли конфігурації середовища (`dev.env`, `ua.env`) у репозиторії. Якщо жодних файлів не знайдено, перевірку пропускає та повертає успішний статус, згадуючи маркер (abie.mdc).
4. Для кожного знайденого файлу конфігурації:
   а. Зчитує вміст файлу. У разі помилки зчитування фіксує збій.
   б. Визначає ім'я середовища на основі імені файлу.
   в. Перевіряє всі внутрішні URL у файлі конфігурації на відповідність визначенню середовища.
   г. Якщо всі URL відповідають, фіксує успішний статус, згадуючи маркер (abie.mdc).
   д. Якщо знайдено невідповідності, фіксує збій для кожного знайденого недоліку, згадуючи маркер (abie.mdc).
5. Повертає загальний код завершення виконання.

- Read-only: не виконує операцій запису (ФС/БД).
