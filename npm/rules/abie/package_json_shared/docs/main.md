## Огляд

Валідує конфігураційний файл `package.json` відповідно до політик, визначених у `abie.mdc`. Перевіряє відповідність файлу встановленим вимогам та повідомляє про виявлені порушення. При відсутності `package.json` сприяє його наявності, забезпечуючи коректність налаштувань, які спираються на `package.json`.

## Поведінка

1. Викликається `lint` для перевірки конфігураційного файлу `package.json` згідно з політиками, визначеними в `abie.mdc`.
2. Якщо `package.json` відсутній, виникає повідомлення про помилку, яке вказує на необхідність створення файлу та додавання `devDependencies['@nitra/abie-shared']` відповідно до `abie.mdc`.
3. В результаті повертається уніфікований звіт про результати лінту, який містить перелік виявлених порушень.

## Публічний API

lint — Відповідає за застосування політики визначення проблем із конфігурацією.
