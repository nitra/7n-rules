## Огляд

Файл проводить оцінку політик, аналізуючи файли у директорії `**/k8s/**/hr.yaml` за допомогою функції `lint`. Ця оцінка спрямована на виявлення потенційних занепокоєнь. Процес сканування виключає будь-які файли, розташовані у піддиректорії `base/`. У результаті генерується єдиний звіт, що фіксує виявлені порушення.

## Поведінка

1. Викликається функція `lint` для оцінки політики з точки зору занепокоєння.
2. Система перевіряє файли за шаблоном `**/k8s/**/base/**/hr.yaml`, при цьому шляхи, що містять `base/`, свідомо ігноруються.
3. У результаті повертається уніфікований звіт про лінт з переліком порушень.

## Публічний API

lint — виявляє положення, що стосуються політики встановлення конфігурації.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Свідомо пропускає шляхи: `base/`.
