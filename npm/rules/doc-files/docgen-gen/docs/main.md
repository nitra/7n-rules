## Огляд

Огляд
Створює технічну документацію шляхом аналізу вихідного коду, генерації тексту, оцінки його якості та управління позиціонуванням захищеного блоку «Призначення» у фінальному документі. (abie.mdc)

Поведінка
splitProtected відокремлює захищений блок «Призначення» від основного вмісту документа, надаючи його тіло та текст без цього блоку.
insertProtected поміщає захищений блок «Призначення» одразу після основного заголовка документа, якщо надано його тіло.
scoreDoc оцінює згенерований документ за заданими критеріями, визначаючи його якість та реєструючи проблеми.
DEFAULT_LOCAL_MODEL встановлює модель, яку використовувати для генерації документа, якщо не вказано іншу.
generateDoc створює повний технічний документ з вихідного файлу, включаючи вихідну оцінку якості.

## Поведінка
