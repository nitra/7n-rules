## Огляд

Модуль виконує логіку перевірки JS-запитань, політик та посилань MDC, ініціюючи повний механізм перевірки. Публічна функція `run` запускає повний процес виконання правила, використовуючи шлях до поточного каталогу.

## Поведінка

1. Викликає повний механізм перевірки, застосовуючи логіку, пов'язану з JS-запитаннями, політиками та посиланнями MDC.
2. При виконанні як автономний інструмент, ініціює повний процес виконання правила, використовуючи шлях до поточного каталогу.

## Публічний API

run — виконує послідовність перевірок: застосування, перевірку JS-зацікавленостей, відповідність політиці та посилання на MDC.
lint — виконує лінтинг, делегуючи складну реалізацію зовнішньому JS-хелперу.
