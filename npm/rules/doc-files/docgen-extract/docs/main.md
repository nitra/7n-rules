1. Аналізує вміст коду для створення детального набору фактів про файл.
2. Визначає словник і визначає, чи є код написаний на JavaScript, TypeScript або TypeScript (ESM).
3. Витягує загальний опис файлу, якщо він присутній у верхньому блоці коментарів.
4. Визначає всі функції, які є публічно доступними (експортованими), разом із їхніми описами та параметрами.
5. Класифікує імпорти на внутрішні, сторонні (npm) та вбудовані (stdlib).
6. Визначає, чи є код:
   - Стійким до винятків (catchesErrors).
   - Здатним працювати без доступу до мережі (readOnly).
