## Огляд

Цей файл аналізує кореляцію між файлами документації та вихідним кодом, використовуючи reverse-map, для визначення, чи стала документація застарілою порівняно з кодом, та чи є вона непов'язаною. Функціональність є read-only: вона не виконує записів у файлову систему чи бази даних. У разі помилок використовується fail-safe механізм, що призводить до повернення `null` замість викидання винятків. При увімкненні механізму виправлення, він може автоматично оновлювати застарілі записи чи видаляти непов'язані файли; у стандартному режимі він лише інформує про виявлені проблеми.

## Поведінка

1. Викликається `lint` для аналізу файлів документації.
2. Визначається наявність застарілої документації та "сирітських" файлів документації (які відносяться до видалених джерел).
3. Якщо виклик є лише для читання (`readOnly` істинно) або відключено автоматичне виправлення (`llmFix` хибно):
   а. Якщо знайдено застарілу документацію, виводиться повідомлення про її застарілість із переліком шляхів до джерел.
   б. Якщо знайдено "сирітські" файли документації, виводиться повідомлення про їхнє існування з переліком шляхів.
4. Якщо увімкнено автоматичне виправлення:
   а. Якщо знайдено застарілу документацію, запускається пакетна генерація для оновлення цієї документації.
   б. Якщо знайдено "сирітські" файли документації, вони видаляються.
5. Після генерації/видалення повторно перевіряється наявність застарілої документації та "сирітських" файлів.
6. Якщо після виправлення знайдено застарілу документацію, виводиться повідомлення про нерозв'язані випадки застарілості.
7. Якщо після виправлення знайдено "сирітські" файли документації, виводиться повідомлення про нерозв'язані випадки їхнього існування.

## Публічний API

lint — виявляє та виправляє порушення у коді;
doc-files — знаходить та виправляє застарілі файли документації.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Перехоплює помилки і не пропускає винятків назовні (fail-safe).
- За певних помилок повертає порожнє значення (напр. `null`) замість винятку.
