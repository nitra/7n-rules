## Огляд

Файл забезпечує генерацію застарілих або відсутніх файлових доків за допомогою docgen-pipeline (локальна/хмарна модель) через функцію fixWorker. Він гарантує відкат даних, реєструючи pre-image кожної доки у ctx.recordWrite до будь-якого запису чи видалення, та виконує очищення від сирітських доків.

## Поведінка

1. Викликається `fixWorker`.
2. Скануються файли для визначення застарілих документів.
3. Якщо знайдені застарілі документи, для кожного з них реєструється його попереднє зображення у `ctx.recordWrite`, а потім запускається генерація пакету документів за допомогою `runGenerationBatch`.
4. Фільтруються порушення, що стосуються сирітських документів (тобто, вихідні файли видалені).
5. Для кожного сирітського документа, якщо є інформація про файл, його попереднє зображення реєструється у `ctx.recordWrite`.
6. Викликається `purgeOrphanedDocs` для видалення сирітських документів.
7. Повертається результат, що містить список дотиків до файлів.
