## Огляд

Перевіряє відповідність файлу `package.json` у поточному контексті заданим конфігураціям, що визначені у package.json. Забезпечує контроль відповідності структури та значень конфігураційних полів проекту. Повертає консолідований звіт про знайдені невідповідності.

## Поведінка

1. Викликаючи функцію `lint`, система перевіряє файли `package.json` у поточному контексті.
2. Перевірка здійснюється на основі конфігурацій, визначених у файлах `package.json`.
3. У результаті повертається єдиний звіт про виявлені порушення.
