## Огляд

Модуль валідує виконання скрипта, коли він запускається як окремий клієнтський інструмент. Валідація здійснюється шляхом порівняння даних, отриманих із JS-запитувань, із визначеними політиками та посиланнями MDC.

## Поведінка

1. Викликається `run` для виконання перевірки.
2. Виконання перевірки застосовує логіку, пов'язану з JS-запитаннями, політикою та посиланнями MDC.
3. Якщо скрипт виконується як окремий клієнтський інструмент, запускається повна логіка правила.

## Публічний API

run — виконує послідовність перевірок: застосовує логіку, обробляє JS-залежності, перевіряє політику та посилання на MDC.
lint — виконує легку перевірку коду, важкі операції перенесено в окремий модуль.
