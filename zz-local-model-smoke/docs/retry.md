## Огляд

Overview
Файл реалізує асинхронну операцію з експоненційним backoff, яка повторюється до успішного завершення або досягнення встановленого ліміту спроб.

Поведінка
1. Визначає максимальну кількість спроб для виконання операції.
2. Визначає початкову затримку для повторів.
3. Ініціює цикл повторів.
4. Виконує саму операцію.
5. Перевіряє, чи досягнуто ліміту спроб після помилки.
6. Якщо ліміт не досягнуто, застосовує затримку, яка зростає експоненціально.
7. Повторює цикл для наступної спроби.
8. Після вичерпання всіх спроб генерує помилку останньої помилки.

## Поведінка

1. Визначається ліміт спроб для виконання операції
2. Визначається базова затримка для повторів
3. Починається цикл повторів
4. Спроба виконання операції
5. У разі помилки перевіряється, чи не досягнуто ліміту спроб
6. Якщо ліміт не досягнуто, виконується затримка, що зростає експоненціально
7. Повторюється цикл для наступної спроби
8. Після вичерпання всіх спроб генерується помилка останньої помилки

## Публічний API

- retry — Повторює асинхронну операцію з експоненційним backoff, поки та не
завершиться успішно або не вичерпається ліміт спроб.
