## ADR Зміна розміру шрифту Markdown Preview

## Context and Problem Statement
Шрифт у вікні Markdown Preview в Cursor IDE виглядав значно більшим, ніж шрифт у терміналі нижче. Користувач попросив зменшити розмір шрифту preview до розміру, співставного з терміналом.

## Considered Options
* Встановити `markdown.preview.fontSize` у Cursor `settings.json`
* Інші варіанти в transcript не обговорювалися.

## Decision Outcome
Chosen option: "Встановити `markdown.preview.fontSize`", because жодних явних налаштувань шрифту (ні `editor.fontSize`, ні `terminal.integrated.fontSize`) у `settings.json` не було — тобто IDE використовувала defaults. Найпряміший спосіб вирівняти preview зі шрифтом терміналу — задати `markdown.preview.fontSize: 13`.

### Consequences
* Good, because розмір тексту у Markdown Preview тепер відповідає стандартному розміру терміналу (~13 px).
* Bad, because значення 13 підібране евристично; якщо пізніше буде задано `terminal.integrated.fontSize` з іншим значенням, preview і термінал можуть знову розійтися.
