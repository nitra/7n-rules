## ADR Зміна шрифту редактора Zed для зручності при астигматизмі

## Context and Problem Statement
Користувач хотів зменшити шрифт у панелі preview та усунути надмірну жирність шрифту в редакторі Zed. Наявний шрифт Menlo не підтримує проміжних вагових варіантів, тому зменшити жирність без заміни шрифту було неможливо. Додатково виявилось, що `ui_font_size` контролює одночасно всі UI-панелі (sidebar, preview, changes), а не лише окремі з них.

## Considered Options
* Зменшити `ui_font_size` (14 → 16 → 18) — впливає на всі UI-панелі разом
* Зберегти Menlo і підібрати `buffer_font_weight` — неефективно, бо Menlo не має проміжних вагових варіантів
* Замінити шрифт на Atkinson Hyperlegible Mono (editor/buffer) + JetBrains Mono Thin (terminal, UI)

## Decision Outcome
Chosen option: "Atkinson Hyperlegible Mono для buffer + JetBrains Mono Thin для terminal і UI", because Atkinson Hyperlegible Mono розроблений Braille Institute спеціально для слабкого зору (широкі літери, чіткі відмінності між схожими символами), а JetBrains Mono має варіант `Thin` що дає необхідно тонке накреслення без CSS-правок.

### Consequences
* Good, because transcript фіксує очікувану користь: користувач підтвердив "стало гарно" після встановлення JetBrains Mono Thin для UI.
* Bad, because `ui_font_size` не можна змінити ізольовано для однієї панелі — зменшення з 21 до 18 зачіпає sidebar, preview і changes panel одночасно.

## More Information
Змінені налаштування у `/Users/vitaliytv/.config/zed/settings.json`:
- `buffer_font_family`: `"Atkinson Hyperlegible Mono"` (editor)
- `terminal.font_family`: `"JetBrains Mono Thin"` (terminal)
- `ui_font_family`: `"JetBrains Mono Thin"` (sidebar, tabs, panels)
- `ui_font_size`: `18` (UI — зменшено з 21)
- `buffer_font_weight`: `400` (залишено, але для Menlo не мало ефекту)

Шрифти встановлені через Homebrew:
```
brew install --cask font-atkinson-hyperlegible-mono
brew install --cask font-jetbrains-mono
```

Zed не підтримує `markdown.preview.fontSize` (як VS Code) та `ui_font_weight` — ці параметри не мали ефекту.
