# Актуалізація посилань на неіснуючий `check-abie.mjs`

**Status:** Accepted
**Date:** 2026-05-23

## Context and Problem Statement

Після реструктуризації `rules/*/js/ → fix/lint/policy` монолітний `check-abie.mjs` було розбито на per-concern файли `fix/<concern>/check.mjs`. Файл `npm/rules/abie/check-abie.mjs` більше не існує, але активні `.mdc`, `.rego` та `.mjs` файли продовжували посилатися на нього — створюючи некоректну документацію та вводячи в оману читача.

## Considered Options

- Оновити всі активні посилання, замінивши `check-abie.mjs` на конкретні `fix/<concern>/check.mjs` або Rego-пакети.
- Інші варіанти в transcript не обговорювалися.

## Decision Outcome

Chosen option: "Оновити всі активні посилання", because файл `check-abie.mjs` більше не існує і посилання на нього вводять читача в оману; після реструктуризації логіка розподілена по `fix/hc_pairing/check.mjs`, `fix/ua_http_route/check.mjs`, `fix/env_dns/check.mjs` та Rego-пакетах `abie.health_check_policy`, `abie.http_route_base`, `abie.base_deployment_preem`.

### Consequences

- Good, because `grep -rn "check-abie" --include="*.mdc" --include="*.mjs" --include="*.rego"` повертає нульовий результат після правок.
- Bad, because transcript не містить підтверджених негативних наслідків.

## More Information
