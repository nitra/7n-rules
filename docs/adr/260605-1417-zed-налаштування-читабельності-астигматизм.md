# Zed — налаштування читабельності при астигматизмі

**Status:** Accepted
**Date:** 2026-06-05

## Context and Problem Statement

Користувач з астигматизмом скаржився, що букви в Zed зливаються. Zed не має власного `letter-spacing`, тому стандартний спосіб розріджування — обрати шрифт і підкрутити суміжні параметри. Додатково, після вирівнювання `buffer_font_size` до 16pt текст у панелі агента виглядав візуально дрібнішим через меншу cap-height sans-serif шрифту порівняно з моноширинним JetBrains Mono при однаковому pt-розмірі.

## Considered Options

* Збільшити лише `buffer_font_size` (масштабувати існуючий шрифт)
* Перейти на JetBrainsMono Nerd Font Mono + комплекс параметрів читабельності (`buffer_line_height: {custom: 1.7}`, `buffer_font_features: {liga: false, calt: false}`, `buffer_font_weight: 450`)
* Виставити `agent_ui_font_size` рівно 16 (ідентично `buffer_font_size`)
* Виставити `agent_ui_font_size: 18`, `agent_buffer_font_size: 16`

## Decision Outcome

Chosen option: "JetBrainsMono Nerd Font Mono + `buffer_line_height: {custom: 1.7}` + `buffer_font_features: {liga: false, calt: false}` + `buffer_font_weight: 450` + `buffer_font_size: 16`; для панелі агента — `agent_ui_font_size: 18`, `agent_buffer_font_size: 16`", because шрифт вже був встановлений у системі і широкий по дизайну (замінює відсутній `letter-spacing`); комплекс параметрів усуває злипання як по горизонталі, так і по вертикалі; sans-serif шрифт панелі агента має меншу cap-height при однаковому pt-розмірі — тому 18pt у панелі дає візуальну рівність із 16pt в редакторі.

### Consequences

* Good, because букви перестали зливатись, рядки отримали «повітря»; JetBrainsMono Nerd Font Mono вже встановлений — жодна додаткова установка не потрібна.
* Good, because після підняття `agent_ui_font_size` до 18 проза і блоки коду в панелі агента виглядають порівняного розміру з текстом редактора.
* Bad, because transcript не містить підтверджених негативних наслідків.

## More Information

Ключ `agent_buffer_font_size` зі значенням за замовчуванням 12 знайдено через `strings` у бінарнику `/Applications/Zed.app/Contents/MacOS/zed`. Шрифт виявлено через `system_profiler SPFontsDataType`. Zed підхоплює зміни live (без перезапуску).

Фінальні зміни у `~/.config/zed/settings.json`:

```json
{
  "buffer_font_family": "JetBrainsMono Nerd Font Mono",
  "buffer_font_size": 16,
  "buffer_font_weight": 450,
  "buffer_line_height": { "custom": 1.7 },
  "buffer_font_features": { "liga": false, "calt": false },
  "agent_ui_font_size": 18,
  "agent_buffer_font_size": 16
}
```
