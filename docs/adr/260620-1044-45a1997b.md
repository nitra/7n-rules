### 🔧 k8s — тільки створити `js/lint.mjs` (CI/pkg — вже чисто)

`lint/lint.mjs` вже є (kubeconform + kubescape). `js/lint.mjs` = один рядок делегата — точно як `python/js/lint.mjs`:

### 🔧 rust — створити `js/lint.mjs`, видалити pkg policy (CI вже без `bun run`)

CI вже запускає `cargo fmt` і `cargo clippy` напряму (не `bun run lint-rust`). Лишилась лише pkg contains-вимога.

`js/lint.mjs` — запускає cargo:
- **fix-режим**: `cargo fmt --all` + `cargo clippy --fix --allow-staged --allow-dirty`
- **read-only**: `cargo fmt --all -- --check` + `cargo clippy --all-targets --all-features -- -D warnings`

Після — видалити `policy/package_json/` (contains: cargo-команди).
