# Runbook n-taze — оновлення залежностей з аналізом major-змін

**Status:** Accepted
**Date:** 2026-05-08

## Контекст

Потрібна стандартизована процедура оновлення npm-залежностей проекту, яка не просто піднімає версії, а й автоматично виявляє breaking changes і рефакторить несумісний код до завершення. Файл `npm/skills/taze/SKILL.md` існував як заглушка без повноцінного workflow.

## Рішення/Процедура

Визначено восьмикроковий workflow у `npm/skills/taze/SKILL.md`:

1. Зафіксувати стартовий стан: `cp package.json package.json.taze-bak`, `cp bun.lock bun.lock.taze-bak`.
2. Запустити `bunx taze -w -r latest` і `bun install`.
3. Порівняти бекапи з оновленими файлами — виявити залежності зі зміною першої значущої цифри semver (major-стрибки).
4. Для кожного major-оновлення зібрати breaking changes: спочатку CHANGELOG/Releases репозиторію модуля, потім `diff -r` між старою версією з кешу bun (`~/.bun/install/cache/<name>@<old>/`) і новою в `node_modules/`, або через тимчасову установку старої версії у `/tmp/taze-old`.
5. Знайти використання зачеплених API у коді проекту через `rg`.
6. Рефакторинг несумісних місць; якщо міграція неоднозначна — залишити TODO з посиланням на CHANGELOG.
7. Прибрати тимчасові файли (`*.taze-bak`).
8. Надати структурований звіт: кількість minor/patch-оновлень, список major-змін, що рефакторено автоматично, що потребує ручного втручання, стан lint/typecheck/test.

## Обґрунтування

Користувач хотів детерміновану процедуру, яка не просто оновлює версії, а гарантує аналіз сумісності — щоб уникнути прихованих регресій від major-стрибків. Збереження бекапів перед оновленням дозволяє точно обчислити diff без покладання на зовнішні реєстри. Пріоритет CHANGELOG над git-diff мінімізує шум при аналізі.

## Розглянуті альтернативи

Неявна альтернатива — просто запустити `taze` і залишити перевірку сумісності розробнику — відхилена на користь автоматичного аналізу та рефакторингу.

## Зачіпає

`npm/skills/taze/SKILL.md`, `npm/package.json`, `npm/CHANGELOG.md`, вихідний код проектів, де викликається skill.
