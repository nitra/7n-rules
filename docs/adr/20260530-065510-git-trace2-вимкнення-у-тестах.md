# Вимкнення git Trace2 у тестах через GIT_TRACE2_EVENT=0

**Status:** Accepted
**Date:** 2026-05-30

## Context and Problem Statement

Git-залежні vitest-тести у `npm/rules/changelog/` та `npm/rules/ga/` таймаутять виключно локально (23 failed, 187с; CI завжди зелена). Кожна git-команда в tmp-репо (`withTmpDir`) успадковує `~/.gitconfig` з `trace2.eventtarget=af_unix:stream:/Users/vitaliytv/.git-ai/internal/daemon/trace2.sock`. Коли даемон `git-ai` деградує, кожен `connect()`/`write()` блокується (~1с wall), а під `pool: 'forks'` десятки паралельних git-операцій перевищують `testTimeout: 5000ms`.

## Considered Options

- `env: { GIT_TRACE2_EVENT: '0' }` у `vitest.config.js`
- `--no-file-parallelism` або `poolOptions.forks.singleFork`
- Підвищити `testTimeout` глобально без усунення trace2-залежності
- Спільний bare-repo fixture замість `git init` у кожному тесті

## Decision Outcome

Chosen option: "`env: { GIT_TRACE2_EVENT: '0' }` у `vitest.config.js`", because env-змінна має пріоритет над config-таргетом і повністю прибирає недетерміновану зовнішню залежність; синтетичні tmp-репо операції не повинні потрапляти в AI-інструментацію. Контрольний прогін із явним `GIT_TRACE2_EVENT=0` дав 36 passed / ~8с — пряме підтвердження причинності.

### Consequences

- Good, because `bun run vitest run rules/changelog/` після фіксу: 36 passed, ~7с (було 23 failed, 187с); відтворювано двічі поспіль.
- Good, because централізований фікс покриває обидва git-залежні suite без змін у тест-коді.
- Good, because повний `bun run test` — 1248 passed | 2 skipped, 0 failures, 20.9с.
- Neutral, because `testTimeout` підвищено до 20000ms як defence-in-depth для APFS/Spotlight-латентності.
- Bad, because деградований стан даемона не вдалось відтворити штучно — механізм патології доведений непрямо.

## More Information

Змінений файл: `npm/vitest.config.js` — `test.env: { GIT_TRACE2_EVENT: '0' }` та `test.testTimeout: 20000`. Глобальна причина: `git config --global trace2.eventtarget` → `af_unix:stream:/Users/vitaliytv/.git-ai/internal/daemon/trace2.sock`. CI не вражена. Git-залежні тести: `npm/rules/changelog/js/tests/consistency/tests/check.test.mjs`, `npm/rules/ga/js/tests/workflows.test.mjs`. Записано у `npm/CHANGELOG.md` під `### Fixed` в релізі `1.31.0`.
