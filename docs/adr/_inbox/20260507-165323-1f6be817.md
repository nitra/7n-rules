## ADR Розщеплення правила `image` на `image-compress` + `image-avif` з авто-міграцією
**Контекст:** У монорепо з декількома Vue-проєктами (адмінка та публічний сайт) виникла потреба вмикати стиснення зображень для всіх пакетів, але AVIF-конвертацію — лише для адмінки (публічний сайт підтримує старіші браузери без AVIF).
**Рішення/Процедура/Факт:** Старе єдине правило `image` розщеплено на два самостійних: `image-compress` (стиснення raster/SVG через `@nitra/minify-image`, без `--avif`) та `image-avif` (генерація `.avif`-двійників, переписування посилань у `.vue`/`.html`, cleanup сиріт). Опт-аут для конкретного workspace-пакета — через `"@nitra/minify-image": { "disable-avif": true }` у відповідному `package.json`. Авто-залежність: `image-avif - [vue, image-compress]`. Авто-міграція `image` → `image-compress + image-avif` вбудована в `mergeConfigWithAutoDetected` через `RULE_MIGRATIONS`, з логом у CLI. Видалено `check-image.mjs`, `image.mdc`, `n-image.mdc`; додано `check-image-compress.mjs`, `check-image-avif.mjs`, відповідні `.mdc`-файли та тести.
**Обґрунтування:** AVIF ще не підтримується всіма браузерами, тому для публічних сайтів його не можна ставити як основне зображення; адмінки натомість орієнтовані на сучасні браузери. Розщеплення правила дає точний контроль без патчінгу глобальних шляхів ігнорування.
**Розглянуті альтернативи:** (1) path-scoped `disable-rules-paths` у `.n-cursor.json`; (2) розширення поля `ignore` об'єктним записом `{ "path": "apps/site", "rules": ["image"] }`. Обидва відхилені на користь нативного package-level opt-out і чіткого розщеплення правил.
**Зачіпає:** `npm/scripts/check-image-compress.mjs`, `npm/scripts/check-image-avif.mjs`, `npm/mdc/image-compress.mdc`, `npm/mdc/image-avif.mdc`, `.cursor/rules/n-image-compress.mdc`, `.cursor/rules/n-image-avif.mdc`, `npm/scripts/auto-rules.mjs` (`RULE_MIGRATIONS`, `migrateRuleIds`, `detectLegacyRuleIds`), `npm/bin/n-cursor.js`, `npm/tests/auto-rules.test.mjs`, `npm/tests/check-image-compress.test.mjs`, `npm/tests/check-image-avif.test.mjs`, `.n-cursor.json`, `CLAUDE.md`, `AGENTS.md`.
