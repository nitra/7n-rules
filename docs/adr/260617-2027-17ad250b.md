### What is OKF?
Open Knowledge Format (OKF) is an open specification for encoding domain knowledge into structured, portable, machine-readable packages called **knowledge bundles**. These bundles are designed to be consumed directly by AI agents and LLM-based systems.

### Why OKF?
Traditional documentation is human-readable but hard for AI to parse. OKF solves this by providing:
- A **consistent structure** AI can reliably navigate
- **Semantic clarity** through typed fields and defined vocabularies
- **Portability** across different AI systems and frameworks

### Knowledge Bundle Structure

```
bundle.okf/
├── okf.json          # Bundle manifest (required)
├── README.md         # Human-readable overview
└── knowledge/
├── overview.md
├── entities.yaml
└── rules.yaml
```

### okf.json Schema

```json
{
"$schema": "https://okflib.org/schema/v1/manifest.schema.json",
"name": "string",
"version": "string (semver)",
"description": "string",
"okf_version": "1",
"entries": [
{
"type": "overview | entity | rule | example | schema",
"path": "knowledge/overview.md"
}
]
}
```

### Entry Types
- **overview**: General description, context, important caveats
- **entity**: Typed description of a data model or concept
- **rule**: Business logic, constraints, or policies
- **example**: Concrete query/answer pairs or use cases
- **schema**: Formal schema (JSON Schema, OpenAPI, etc.)

### Minimal Example

```json
{
"name": "acme-orders",
"version": "1.0.0",
"description": "Order management system knowledge bundle",
"okf_version": "1",
"entries": [
{ "type": "overview", "path": "knowledge/overview.md" },
{ "type": "entity", "path": "knowledge/entities.yaml" },
{ "type": "rule", "path": "knowledge/rules.yaml" }
]
}
```

### Libraries
- **Python**: `pip install okflib`
- **JavaScript/TypeScript**: `npm install okflib`

### Quick Start (Python)

```python
from okflib import Bundle

bundle = Bundle.load("path/to/bundle.okf")
context = bundle.context()  # Returns a string for injection into AI prompts
```

### Quick Start (JavaScript)

```javascript
import { Bundle } from 'okflib';

const bundle = await Bundle.load('path/to/bundle.okf');
const context = bundle.context(); // Returns string for AI prompt injection
```

### Integration Note
OKF bundles are **format-agnostic** — the `context()` method returns a plain string that can be injected into any LLM system prompt, regardless of the AI provider.
