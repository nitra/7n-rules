## Considered Options
* Залишити як internal subcommands (subprocess-based)

Chosen option: "Прямі функції без subprocess", because усуває зайвий fork-overhead; orchestrator/t0/post-tool-use-fix/lint-orchestrate importe напряму; `_fix-check` і `fix-t0` cases видалено з `bin/n-cursor.js`.
