# Систематичне скорочення `.mdc`-правил: видалення нерезультативного тексту

**Status:** Accepted
**Date:** 2026-05-17

## Context and Problem Statement

Файли правил у `npm/rules/**/*.mdc` накопичили пласт пояснювального тексту (мета-блоки активації, інструкції CLI, VS Code блоки, розділи `## Перевірка`, описи зовнішніх інструментів), що збільшувало контекст при підвантаженні правил без впливу на поведінку агента.

## Considered Options

- Систематично прибрати п'ять класів нерезультативного тексту з усіх правил
- Прибирати лише найгрубіші блоки вибірково
- Залишити файли без змін

## Decision Outcome

Chosen option: "Систематично прибрати п'ять класів нерезультативного тексту", because `.mdc` є виключно AI-контекстом; CLI n-cursor дізнається про концерни виключно з файлової структури — `.mdc` для цього не читається.

### Consequences

- Good, because ~314 рядків (~7,6%) видалено з 23 файлів; контекстний тягар зменшено без втрати семантичних директив.
- Neutral, because `vue.mdc`, `php.mdc`, `text.mdc` навмисно пропущені через ризик суперечностей із check-скриптами.
- Bad, because transcript не містить підтвердження негативних наслідків.

## More Information

П'ять видалених класів: (1) секції `## Перевірка` (18 файлів); (2) `.vscode/extensions.json` і `.vscode/settings.json` (8 файлів); (3) `brew install` / `apt-get` (5 файлів); (4) описи зовнішніх інструментів; (5) активаційна мета, міграційний контекст, дублікати.
Зачеплені: `security`, `rego`, `adr`, `style-lint`, `image-compress`, `image-avif`, `nginx-default-tpl`, `graphql`, `docker`, `bun`, `k8s`, `js-lint`, `ga`, `hasura`, `capacitor`, `changelog`, `npm-module`, `ci4`, `js-run`, `js-bun-db`, `js-bun-redis`, `tauri`, `js-mssql`, `abie`.

## Update 2026-05-17

Додатковий шостий патерн — дублікати одного й того ж правила (наприклад `bun.mdc` мав тричі той самий список). Borderline-блоки залишені: `adr.mdc` ENV-таблиця, LLM fallback. `.vscode`-конфіги зберігались там, де є реальна policy-перевірка (наприклад `graphql/policy/vscode_extensions/` перевіряє `"graphql.vscode-graphql"`); видалялись лише де відповідного policy-концерну немає.
