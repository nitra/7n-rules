## Секція 1: Схема `meta.json`

Два окремі JSON-схеми (E2), що живуть у `npm/schemas/`:

**`skill-meta.schema.json`** (для `npm/skills/<id>/meta.json`):
```json
{
"type": "object",
"required": ["worktree"],
"additionalProperties": false,
"properties": {
"auto": { "oneOf": [
{ "type": "string" },
{ "type": "array", "items": { "type": "string" } }
]},
"worktree": { "type": "boolean" }
}
}
```

**`rule-meta.schema.json`** (для `npm/rules/<id>/meta.json`):
```json
{
"type": "object",
"additionalProperties": false,
"properties": {

{ "type": "array", "items": { "type": "string" } }
]}
}
}
```

`auto` — опційне в обох (скіли без умови автоактивації / правила, що не автовизначаються, просто не мають поля).
