## Що змінилось

**`npx @nitra/cursor fix`** тепер — повноцінний автономний оркестратор:

```
bun n-cursor.js fix [bun ga rego]
│
├─ iter 1: checkRules() → T0-auto → recheck
│          → якщо 0 failed: done ✅
│          → haiku LLM per rule → recheck
│
├─ iter 2: haiku LLM
│
├─ iter 3: sonnet (ескалація)
│
└─ без ANTHROPIC_API_KEY → T0-auto only, звіт unresolved
```

**SKILL.md** — зведено до одного рядка:
```bash
n_cursor_npx fix
```

**Нові файли:**
- `npm/skills/fix/js/orchestrator.mjs` — convergence-loop, ескалація haiku→sonnet
- `npm/skills/fix/js/llm-worker.mjs` — Claude SDK з tools `read_file`/`write_file`/`run_command`
