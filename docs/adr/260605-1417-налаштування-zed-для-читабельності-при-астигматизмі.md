## ADR Налаштування Zed для читабельності при астигматизмі

## Context and Problem Statement
Користувач має астигматизм — букви у редакторі Zed зливались, що утруднювало читання. Zed не підтримує пряме налаштування `letter-spacing`, тому потрібно було знайти рівнозначні важелі.

## Considered Options
* Збільшити міжрядковий інтервал (`buffer_line_height`) і розмір шрифту
* Замінити шрифт на широкий моноширинний (JetBrains Mono, Comic Code, OpenDyslexic Mono, IBM Plex Mono, Hack, Iosevka)
* Вимкнути лігатури (`liga`, `calt`) щоб символи не зливались у єдиний гліф
* Збільшити товщину ліній (`buffer_font_weight`)

## Decision Outcome
Chosen option: "Комбінація: JetBrains Mono + line-height 1.7 + вимкнені лігатури + font-weight 450 + розмір 16pt", because шрифт JetBrains Mono вже був встановлений у системі, а поєднання широкого трекінгу шрифту з розрідженими рядками і без лігатур дає найбільший ефект проти злипання без потреби у невбудованому `letter-spacing`.

### Consequences
* Good, because transcript фіксує очікувану користь: букви перестали зливатись, рядки отримали «повітря», JetBrainsMono Nerd Font Mono вже встановлений — жодна додаткова установка не потрібна.
* Bad, because transcript не містить підтверджених негативних наслідків.

## More Information
Змінені ключі в `~/.config/zed/settings.json`:
```json
{
"buffer_font_family": "JetBrainsMono Nerd Font Mono",
"buffer_font_size": 16,
"buffer_font_weight": 450,
"buffer_line_height": { "custom": 1.7 },
"buffer_font_features": { "liga": false, "calt": false }
}
```
Шрифт виявлено через `system_profiler SPFontsDataType`. Zed підхоплює зміни live (без перезапуску).

## ADR Окреме збільшення шрифту панелі агента Zed

## Context and Problem Statement
Після вирівнювання `buffer_font_size` до 16pt текст у панелі агента виглядав візуально дрібнішим, ніж у редакторі. Zed має незалежні налаштування для панелі агента (`agent_ui_font_size`, `agent_buffer_font_size`), дефолтне значення яких — 12pt.

## Considered Options
* Виставити `agent_ui_font_size` і `agent_buffer_font_size` рівно 16, як у редакторі
* Виставити `agent_ui_font_size` вище (18), бо sans-шрифт інтерфейсу при однаковому pt-розмірі виглядає візуально меншим за моноширинний JetBrains Mono

## Decision Outcome
Chosen option: "`agent_ui_font_size`: 18 + `agent_buffer_font_size`: 16", because sans-serif шрифт панелі агента має меншу висоту літер (cap-height) при тому самому pt-розмірі, ніж моноширинний JetBrains Mono у редакторі — тому 18pt у панелі дає візуальну рівність із 16pt в редакторі.

### Consequences
* Good, because transcript фіксує очікувану користь: візуальний розмір тексту у панелі агента і в редакторі вирівнюється.
* Bad, because transcript не містить підтверджених негативних наслідків.

## More Information
Ключ `agent_buffer_font_size` зі значенням за замовчуванням 12 знайдено через `strings` у бінарнику `/Applications/Zed.app/Contents/MacOS/zed`. Змінені ключі в `~/.config/zed/settings.json`:
```json
{
"agent_ui_font_size": 18,
"agent_buffer_font_size": 16
}
```
