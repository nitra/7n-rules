## Decision Outcome

## ADR Перейменування CLI-команди `check` → `fix` зі spawn-wrapper

## Context and Problem Statement
CLI-команда `npx @nitra/cursor check <rule>` не узгоджувалася з новим іменем файла `rules/<id>/fix.mjs`. Після введення dual-mode `fix.mjs` постало питання щодо того, як CLI-команда співвідноситься зі standalone-запуском.

## Considered Options
* Зробити `fix` spawn-wrapper: `case 'fix'` → `spawn('bun', ['rules/<id>/fix.mjs'])`; `case 'check'` — deprecated alias з warning
* Залишити CLI як є (без rename)
* Інші варіанти в transcript не обговорювалися.

## Decision Outcome
Chosen option: "CLI `fix` як spawn-wrapper + deprecated alias `check`", because це узгоджує команду з ім'ям файла та робить `npx @nitra/cursor fix abie` семантично рівнозначним `bun rules/abie/fix.mjs`. Backward-compat alias `check` зберігає сумісність із наявними скриптами.

### Consequences
* Good, because transcript фіксує очікувану користь: `bun npm/bin/n-cursor.js fix text` виконує повний run; `bun npm/bin/n-cursor.js check text` друкує deprecation warning та продовжує роботу.
* Bad, because transcript не містить підтверджених негативних наслідків.

## More Information
