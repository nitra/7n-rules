# План: розкатка перегенерації файлових док local-only конвеєром

**Дата:** 2026-06-11
**Статус:** до виконання (фази 1–2 — будь-коли на вільній машині; фаза 3 — окрема сесія)
**Контекст:** Крок 0 local-only конвеєра злито (коміти `d6c23596`, `e2232d07`); дизайн — спек `docs/superpowers/specs/2026-06-10-doc-files-local-only-pipeline.md`, ADR `260610-2228`; опис алгоритму — `docs/doc-files-skill.md`.

## Стан на момент фіксації

- **224** кодових файли-кандидати: **158 fresh / 66 stale (беклог) / 0 mismatch**.
- Міграцію CRC завершено: 126 git-актуальним докам застамповано frontmatter; 73 git-відсталі доки + 3 без док свідомо лишено stale (штампувати їх — означало б збрехати про актуальність).
- Хронологія попередніх замін док: хвилі 06-04 (13), 06-05 (94), 06-06 (48), 06-08 (28) — покоління до-CRC конвеєрів (ollama/cloud); 37% з них тихо відстали від коду з лагом до 5 днів. Дрейф більше не накопичується — Stop-гейт ловить кожну правку.

**Пілот 12 файлів (06-11, авто-пілот):**

- 21.4 хв на 12 файлів ≈ **1.8 хв/файл** (збігається з прогнозом);
- **10 OK** (усі det-score=100), **0 degraded**, **2 ✗** — `curl exit 18` (transient обрив стріму, ймовірно під тиском пам'яті; пережив 3 ретраї `callOmlx`). Обидва (`npm/lib/models.mjs`, `npm/rules/changelog/js/consistency.mjs`) лишились у беклозі — система самоузгоджена;
- нагадування: det-score міряє **форму**, не правдивість змісту — якість заміни оцінюється очима у Фазі 1.

## Фаза 5 — Оптимізації за спостереженнями пілота (опційно)

- [ ] **Content-addressed дедуп генерації:** однакові джерела мають однаковий CRC (пілот: `capacitor/fix.mjs` і `ci4/fix.mjs` → один crc `12fc1644`) — генерувати доку один раз на унікальний CRC і штампувати всім близнюкам. На 224 файлах із кластерами однакових `fix.mjs`-стабів — економія десятків викликів.
- [ ] **Ретрай файлу після `curl exit 18`:** зараз ретраїться лише HTTP-рівень; обрив посеред секції валить весь файл (✗). Варіанти: 1 повтор файлу в батчі або лишити як є (CRC тримає stale — наступний прогін добере). Рішення — за частотою у Фазі 2.
- [ ] Уже винесено в чіпи окремих сесій: cspell-марафон `lint-text` (~3000 слів) і дедуплікація AST-сканерів правил на `ast-scan-utils`.
