## Огляд

Файл реалізує механізми для реакції на помилки, пов'язані з нестачею оперативної пам'яті на локальному model-сервері. Він використовує `MEMORY_ERROR_RE` для ідентифікації таких відмов та кидає Error через `failOnMemoryGuard` при виявленні браку пам'яті.

## Поведінка

MEMORY_ERROR_RE — Регулярний вираз для ідентифікації відмов локального model-сервера через брак оперативної пам'яті.
failOnMemoryGuard — При отриманні повідомлення про відмову через брак пам'яті, виводить тіло запиту в стандартний вивід та кидає Error.

## Публічний API

MEMORY_ERROR_RE — Знаходить відмову локального сервера моделі (наприклад, oMLX) через нестачу пам'яті при обробці промпта.
failOnMemoryGuard — При отриманні повідомлення про відмову через механізм захисту пам'яті виводить тіло запиту на стандартний вивід і кидає Error; інакше нічого не робить.
