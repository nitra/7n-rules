## Огляд

Модуль ініціює процес виправлення порушень правил. Він створює відповідний промпт для агента через функцію `buildFixPrompt` та запускає цикл взаємодії з агентом за допомогою `runAgentFix`. Процес працює з механізмом перехоплення помилок (fail-safe), гарантуючи відсутність винятків на виході.

## Поведінка

Поведінка:
buildFixPrompt готує текстовий промпт, що містить інструкції для агента щодо виправлення порушення правила.
runAgentFix виконує повний агентний цикл для спроби виправлення порушення правила, включаючи взаємодію з інструментами, застосування патча та фіксацію телеметрії.

## Публічний API

buildFixPrompt — формує інструкцію для виправлення проблеми, включаючи відповідне правило, описане порушення та можливий відгук з попередньої невдалої спроби, з додатковою директивою щодо кроків перед редагуванням та самоперевірки.
runAgentFix — виконує єдину спробу виправлення коду з боку агента для заданого правила, використовуючи вказані параметри (модель, рівень, зворотний зв'язок, тощо) та ін'єкції для тестування.

## Гарантії поведінки

- Перехоплює помилки і не пропускає винятків назовні (fail-safe).
