## Огляд

Жива обкатка композитної драбини з README крейта: Cursor → Codex → local → cloud. Кожна ланка — реальний виклик, не мок. Запуск: `cargo run --example ladder_live`.

## Поведінка

Проходить рунги по черзі з одним промптом («Скажи рівно одне слово: працює»): ACP Cursor → ACP Codex → локальний `Tier::Min` через omlx (`http://127.0.0.1:8000/v1/`, опційний ключ з `OMLX_API_KEY`) → хмарний `Tier::Max`. Провал рунга друкується в stderr (`⏭️ <рунг> провалився: <помилка>`), і драбина йде далі. Перший успішний рунг завершує роботу звітом: назва рунга, загальний час від старту, текст відповіді. Якщо провалились усі рунги — `❌ вся драбина провалилась` і вихід з кодом 1.
