## Огляд

Файл вибирає, який ACP-агент запускати, і спілкується з уже залогіненим локальним агентським CLI через stdio/JSON-RPC. Це шлях для доступу до потужних моделей за особистою підпискою: `agent login` / `codex login` має бути виконано заздалегідь власником, а крейт не виставляє API-ключів для цього маршруту. Якщо CLI не залогінений підпискою, виклик завершується невдало, а рішення про перехід на [`crate::local_cloud`] лишається викликачеві за fail-fast принципом. За певних помилок повертає порожнє значення замість винятку.

## Поведінка

AcpAgentKind визначає, який саме ACP-агент буде запущено, і відсікає застарілі містки на користь офіційних шляхів для Cursor та Codex.

one_shot_acp бере вибраний AcpAgentKind, готує його ACP-спеку й передає далі в транспортний шар разом із prompt та робочим каталогом виклику. Результат повертається як повний текст відповіді від уже залогіненого локального агента; якщо підписка не активна або агент недоступний, запит провалюється без fallback-поведінки всередині цього шляху, і рішення про перехід на інший бекенд лишається викликачеві.

## Публічний API

- AcpAgentKind — Ціль ACP-підключення. Список відкритий — нові агенти додаються нових варіантом і `spec()`, протокол той самий для всіх.
- one_shot_acp — Один виклик через ACP: спавнить агента, відкриває сесію в `cwd`, надсилає `prompt`, читає повний текст відповіді до кінця ходу.  `cwd` — явний, а не [`std::env::current_dir`]: викликач (напр. napi-міст в один процес із Node) має свій власний робочий каталог проєкту, який не обов'язково збігається з cwd самого хост-процесу.  Тонкий фасад над [`transport`]: спавн/init/session — спільний шар, який буде перевикористаний публічним session-API (наступна задача) поверх того самого транспорту.  # Errors [`CascadeError::Provider`] — агент не встановлений, не залогінений підпискою, чи процес завершився з помилкою.

## Гарантії поведінки

- Власних операцій запису (ФС/БД) у файлі немає; виклики імпортованих модулів можуть писати.
- За певних помилок повертає порожнє значення (напр. `null`) замість винятку.
