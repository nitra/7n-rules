Огляд

Поведінка

1. Визначає області для сканування тестових файлів, включаючи їх розташування поряд із кодом або в інтеграційних наборах у корені.

4. Збирає звіт про покриття коду, використовуючи `v8` як постачальника та генеруючи звіти у форматах `lcov` та `text-summary`.
