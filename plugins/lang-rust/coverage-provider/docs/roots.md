## Огляд

`findRustRoots` надає Rust-специфічне визначення коренів проєкту для coverage-виміру за маніфестом `Cargo.toml`. Файл існує як тонка обгортка над спільною lib концерну coverage, щоб Rust-пошук використовував єдину поведінку виявлення коренів.

## Поведінка

1. `findRustRoots` визначає каталоги Rust-проєктів, які мають брати участь у coverage-вимірі.
2. Для пошуку використовується ознака Rust-проєкту — наявність маніфесту `Cargo.toml`.
3. Результат охоплює корінь перевірки та теки першого рівня, у яких знайдено Rust-маніфест.
4. Функція лише делегує пошук спільному механізму coverage-інфраструктури й не змінює файлову систему.

## Публічний API

- findRustRoots — Rust-корені під `cwd` (корінь + перший рівень тек із Cargo.toml).

## Гарантії поведінки

- Власних операцій запису (ФС/БД) у файлі немає; виклики імпортованих модулів можуть писати.
