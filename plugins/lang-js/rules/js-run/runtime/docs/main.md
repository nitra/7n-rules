## Огляд

Цей файл забезпечує відповідність проєкту затвердженим правилам `js-run.mdc`, спираючись на конфігурації `jsconfig.json` та `package.json`. Він перевіряє конфігурації, імпорти, неймінг, змінні середовища та таймери для кожного пакета, ігноруючи перевірку для frontend-пакетів, що використовують `vite`.

## Поведінка

Поведінка:
pgWriteContract: Не є публічною функцією.
main: Перевіряє відповідність проєкту правилам `js-run.mdc` для кожного workspace-пакета, пропускаючи перевірку для frontend-пакетів, що використовують `vite`. Для пакетів відбувається валідація конфігурацій, імпортів, неймінгу файлів, використання змінних середовища та таймерів.
