## Огляд

Задає Vitest-конфіг для Vue-бібліотеки зі Storybook-скоупом. Файл існує, щоб централізувати тестове середовище для бібліотеки та перевірок, пов’язаних зі Storybook, в одному конфігураційному місці.

## Поведінка

1. Формує єдиний тестовий профіль для Vue-компонентної бібліотеки, який поєднує базові налаштування збірки з правилами запуску Vitest.

2. Визначає звичайний unit-набір як окремий named project, щоб тести бібліотеки запускалися ізольовано від Storybook-сценаріїв.

3. Додає окремий Storybook-набір, який перевіряє історії компонентів у browser-mode через Chromium.

4. Обмежує Storybook-перевірки свідомо підставленим шаблонним шляхом до stories, який має бути замінений під час генерації конфігу.

5. Підключає підготовчий Storybook-файл перед запуском browser-перевірок, щоб середовище відповідало умовам рендерингу історій.

6. Налаштовує покриття для тестів бібліотеки у форматах, придатних і для локального перегляду, і для CI-звітності.

7. Залишає mutation-testing поза цим конфігом, бо для нього очікується окремий ізольований профіль без Storybook browser-mode projects.

## Гарантії поведінки

- Власних операцій запису (ФС/БД) у файлі немає; виклики імпортованих модулів можуть писати.
