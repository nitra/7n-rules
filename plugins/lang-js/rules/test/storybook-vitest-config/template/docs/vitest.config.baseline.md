## Огляд

Файл збирає спільну тестову конфігурацію для Vue-бібліотеки в Storybook-контексті: окремо для звичайних unit-тестів і окремо для storybook-сценаріїв. Він існує, щоб один набір використовувався для ізольованих DOM-тестів у unit-режимі, а інший — для перевірки історій у Storybook-оточенні.

## Поведінка

1. Об’єднує базову конфігурацію проєкту з канонічним тестовим профілем для бібліотеки Vue-компонентів у контексті Storybook.
2. Визначає окремий unit-набір для звичайних тестів і окремий storybook-набір для візуально-інтеграційних сценаріїв.
3. Для unit-набору запускає тести в середовищі, придатному для DOM-логіки без браузера, і збирає покриття у форматах для звіту та короткого підсумку.
4. Для storybook-набору підключає перевірку історій із каталогом Storybook, вмикає browser-mode, працює без віконного інтерфейсу та обмежується Chromium.
5. Для storybook-набору підвантажує окрему підготовку середовища з сусідньої storybook-налаштування.
6. Не запускає browser-mode для unit-набору, щоб зберегти сумісність із окремим ізольованим mutant-testing сценарієм, який використовує той самий unit-набір без browser projects.
