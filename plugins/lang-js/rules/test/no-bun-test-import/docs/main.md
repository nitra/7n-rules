## Огляд

Огляд
Цей компонент аналізує тестові файли проекту для модернізації тестового фреймворку. Він ідентифікує імпорти з `bun:test` та перевіряє їхню сумісність для потенційного переходу на `vitest`, згенерувавши відповідні порушення для виправлення.

Поведінка
findBunTestImports виявляє імпорти з `bun:test` у коді, визначаючи, які з них можуть бути безпечно замінені на `vitest`.
lint перевіряє тестові файли на наявність імпортів з `bun:test`, генеруючи порушення, якщо імпорт не може бути автоматично виправлений (test.mdc).

## Поведінка

Поведінка
findBunTestImports знаходить усі імпорти з `bun:test` у коді, визначаючи, які з них можуть бути безпечно замінені на `vitest`.
lint перевіряє тестові файли на наявність імпортів з `bun:test`, генеруючи порушення, якщо імпорт не може бути автоматично виправлений (test.mdc).

## Публічний API

findBunTestImports — шукає усі імпорти з `bun:test` у коді файлу, щоб уникнути повторного аналізу.
lint — гарантує, що тестові файли не імпортують модулі з `bun:test`, оскільки Vitest не може їх знайти.
