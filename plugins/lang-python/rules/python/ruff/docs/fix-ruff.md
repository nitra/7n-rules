## Огляд

T0-autofix для `python/ruff` забезпечує детерміновані виправлення та форматування файлів проєкту (`ruff check --fix` та `ruff format`), якщо інструмент `uv` доступний. Його призначення — стандартизувати код, порівнюючи зміни, внесені через автоматичні інструменти, з початковим вмістом.

## Поведінка

Поведінка
1. Перевіряє, чи доступна команда `uv`. Якщо ні, жодних дій не виконує.
2. Визначає список усіх файлів `.py` у проєкті, використовуючи інструмент `git`.
3. Зберігає поточний вміст усіх знайдених `.py` файлів.
4. Виконує команду `ruff check --fix` у всіх файлах проєкту, щоб автоматично виправити порушення.
5. Виконує команду `ruff format` у всіх файлах проєкту, щоб застосувати стандарт форматування.
6. Порівнює поточний вміст файлів із вмістом до виправлення.
7. Якщо вміст файлу змінився, цей файл позначається як змінений.
8. Якщо були змінені файли, результат повідомляється, а ці файли реєструються як змінені.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Перехоплює помилки і не пропускає винятків назовні (fail-safe).
- За певних помилок повертає порожнє значення (напр. `null`) замість винятку.
